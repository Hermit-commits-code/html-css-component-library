# HTML/CSS Component Library

A handcrafted component library built with semantic HTML and modern CSS.

This project is a foundations build for practicing reusable UI patterns, responsive layouts, accessibility basics, and visual consistency before moving into Tailwind, React, and component-driven frameworks.

## Current Status

In progress.

## Tech Stack

- HTML
- CSS
- Git
- GitHub

## Goals

- Practice semantic HTML
- Build reusable UI components
- Improve responsive design skills
- Strengthen Flexbox and Grid fundamentals
- Develop a consistent spacing and typography system
- Document accessibility considerations for each component

## Planned Components

- Buttons
- Cards
- Forms
- Navigation bars
- Hero sections
- Feature grids
- Pricing sections
- Testimonials
- Modal layouts

## Project Philosophy

This is not an npm package or production design system yet.

The goal is to build strong frontend fundamentals by hand before relying on frameworks or component libraries.
